Pareto chart

The LineSeries on this pareto chart is created using a custom aggregation function. The data is grouped by client and each client's contribution to total revenue is calculated using group.postAggregation(aggregateFunction). The result of the custom aggregation function is a Percentage property on each group which can be accessed by the valueFunction() of the LineSeries.

function customAggregation(group) { var aggregateFunction = function(grouping) { var total = 0; grouping.getData() .forEach(function(d) { total += d.value.CurrentRevenue.Sum; }); grouping.getData() .forEach(function(d) { d.value.Percentage = d.value.CurrentRevenue.SumCumulative / total; }); }; group.postAggregation(aggregateFunction); }
