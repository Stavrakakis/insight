InsightJs allows axis tick labels and series tooltips to be formatted using axis.tickLabelFormat(formatFunction) and series.tooltipFormat(formatFunction).

In both cases formatFunction is a function that accepts a value argument and returns a formatted string. InsightJs provides a set of useful format functions in the insight.Formatters namespace. See API Docs for details of format functions provided.
