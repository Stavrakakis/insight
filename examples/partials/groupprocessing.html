This pareto chart shows the cumulative contribution of a list of clients to total revenue, as a percentage of total revenue.

The dataset in this example is an insight.DataSet containing employee objects with Client and CurrentRevenue properties.

The client group is created like this: var clientGroup = dataset.group('clients', function(d) { return d.Client; }) .sum(['CurrentRevenue']) .cumulative(['CurrentRevenue.sum']) .orderingFunction(function(a, b) { return b.value.CurrentRevenue.sum - a.value.CurrentRevenue.sum; });

This causes the data to be grouped by Client with each client group containing a CurrentRevenue.sum property and a CurrentRevenue.sumCumulative property.

Calling clientGroup.postAggregation(someFunction) on a group allows further processing to be done on the group. This pareto chart uses the CurrentRevenue.sum and CurrentRevenue.sumCumulative properties of each client, in order to calculate the percentage that the client contributed to total revenue and store it in the Percentage property for each client: var aggregateFunction = function(clientGrouping) { var total = 0; clientGrouping.extractData() .forEach(function(client) { total += client.value.CurrentRevenue.sum; }); clientGrouping.extractData() .forEach(function(client) { client.value.Percentage = client.value.CurrentRevenue.sumCumulative / total; }); }; clientGroup.postAggregation(aggregateFunction);
